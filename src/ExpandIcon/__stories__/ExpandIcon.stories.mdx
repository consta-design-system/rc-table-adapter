# ExpandIcon

```tsx
import { ExpandIcon } from '@consta/rc-table-adapter/ExpandIcon'
```

Кастомизированая под Consta иконка для таблицы.

## Пример

```tsx
const Example = () => {
  const prefix = cnRcTable({
    size: 's',
    zebraStriped: 'odd',
    borderBetweenColumns: true,
    borderBetweenRows: true,
    verticalAlign: 'center',
    headerVerticalAlign: 'center',
    prefixCls: 'CustomTable',
  })

  return <RCTable expandIcon={ExpandIcon} prefixCls={prefix} data={data} columns={columns} />
}
```
