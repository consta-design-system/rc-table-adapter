# ExpandIcon

```tsx
import { ExpandIcon } from '@consta/rc-table-adapter/ExpandIcon';
```

Эта иконка нужна, чтобы показывать скрытые элементы таблицы и скрывать их. Стилизована для дизайн-системы Consta.

## Пример

```tsx
const Example = () => {
  const prefix = cnRcTable(
    {
      size: 's',
      zebraStriped: 'odd',
      borderBetweenColumns: true,
      borderBetweenRows: true,
      verticalAlign: 'center',
      headerVerticalAlign: 'center',
    },
    ['CustomTable'],
  );

  return (
    <RCTable
      expandIcon={ExpandIcon}
      prefixCls={prefix}
      data={data}
      columns={columns}
    />
  );
};
```
