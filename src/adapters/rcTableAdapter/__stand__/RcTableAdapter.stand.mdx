# rcTableAdapter

```tsx
import { rcTableAdapter } from '@consta/rc-table-adapter/rcTableAdapter';
```

Этот адаптер стилизует таблицу [rc-table](https://github.com/react-component/table) для [дизайн-системы Consta](https://consta.design/), кастомизирует иконки и устанавливает значения по умолчанию для некоторых свойств.
