# Быстрый старт

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/rc-table` или `npm install @consta/uikit @consta/rc-table`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/rc-table` были вложены в компонент `Theme`.

#### Всë!
