# rcTableAdapter | Дизайн-система Consta

Адптер для таблицы из библиотеки rc-table. Стилизует таблицу для [дизайн-системы Consta](https://consta.design/).

## Быстрый старт

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/rc-table-adapter` или `npm install @consta/uikit @consta/rc-table-adapter`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты были вложены в [компонент `Theme`](http://uikit.consta.design/?path=/docs/components-theme--playground).

#### Создайте стилизованную таблицу

Есть два варианта:

- Подключите миксин [RcTable](?path=/docs/rctable--page) — он добавит таблице стили из Consta. Если нужно скрыть часть таблицы, используйте иконку [ExpandIcon](?path=/docs/expandicon--page).
- Подключите адаптер [rcTableAdapter](?path=/docs/utils-rctableadapter--playground), чтобы добавить стили, иконки и значения по умолчанию для некоторых свойств.

#### Всë!
