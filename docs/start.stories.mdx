# useRcTableAdapter | Дизайн-система Consta

Этот хук адаптирует таблицы из библиотеки [rc-table](https://github.com/react-component/table) для [дизайн-системы Consta](https://consta.gazprom-neft.ru/).

## Быстрый старт

#### Установите пакет и зависимости

`yarn add @consta/rc-table-adapter` или `npm install @consta/rc-table-adapter`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты были вложены в компонент `Theme`.

#### Всë!
